Lecture 1: Introduction to HTML5 (Interactive)
1. What is HTML5?
Definition:
HTML (HyperText Markup Language): The standard language for creating webpages.
HTML5: The latest version, introducing new features for multimedia, semantics, and better integration with modern web technologies.

2. Why is HTML5 Important?
Semantics: Improved readability for humans and machines (e.g., <article>, <section>).
Multimedia: Simplifies embedding videos and audio without plugins.
Compatibility: Works across all modern browsers.
APIs: Integrates with powerful APIs like Geolocation and Web Storage.

3. Explanation of Key Elements
<!DOCTYPE html>

Declares the document as HTML5.
Ensures browsers interpret the page correctly.
<html lang="en">

Starts the HTML document.
lang="en" specifies the language as English.
